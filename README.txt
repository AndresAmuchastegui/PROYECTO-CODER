Bienvenido al sitio web de Estudio Dignity
Este sitio web fue creado para el proyecto final del curso de desarrollador web de CoderHouse. Esperamos que encuentres todo lo que estás buscando y que te diviertas explorando el sitio.



Características

	- Este sitio fue desarrollado por Andrés Amuchástegui, como primer proyecto de desarrollo. 

	- La web es fácil de utilizar e intuitiva para el usuario

	- Se buscó crear un sitio que simule un estudio contable, impositivo y jurídico 

	- El contenido es 100% ficticio en todos sus apartados

	- Los textos que se pueden ver en las distintas secciones fueron generados por ChatGPT con el objetivo de ir incorporando nuevas herramientas (la programación fue 100% manual).

	- Se utilizaron las distintas herramientas aprendidas en el curso (SASS, Bootstrap, Grid, Git, GitHub, SEO, HTML, CSS, etc.)



Cómo usar el sitio

	- El sitio se puede navegar desde el inicio y por todas las secciones de la barra de navegación.

	

Gracias por visitar Estudio Dignity. ¡Disfruta tu estancia!
